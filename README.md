# Big Data Analytics
=====================
`keys["github"]["access_token"]`
keys is a json file containing the API keys with this format:
```
{
  "twitter" : {
    "consumer_key" : "YOUR_KEY",
    "consumer_secret" : "YOUR_SECRET",
    "..." : "..."
  }
  "github" : {
    "access_token": "YOUR_TOKEN"
  }
}
```
